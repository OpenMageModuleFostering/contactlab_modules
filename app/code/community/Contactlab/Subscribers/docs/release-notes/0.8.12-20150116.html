Async Unique Constraint table update

The Unique Constraint table will be populated within an asyncronous process in cron.

During plugin install or if a user run this function from the task management, a new task will be created, avoiding memory and max process time issues.

Other options

  • Added Check UK Table consistency before every export option to configuration. Check the consistency of UK table after every export (true by default). If the UK table isn't consistent, the export won't begin.
